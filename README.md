# 📱 Caza Insectos

Juego móvil desarrollado con **React Native** y **Expo**, donde el usuario debe capturar insectos virtuales que aparecen en la pantalla. Los puntajes se almacenan en una base de datos externa (Supabase).

---

## 🎯 Objetivo
Crear la interfaz gráfica completa de una aplicación de videojuegos, enfocándose en componentes visuales, estilos y navegación entre pantallas, sin lógica de negocio avanzada ni consumo de APIs externos.

---

## 📝 Descripción
Este proyecto corresponde a un taller práctico de la asignatura **Aplicaciones Móviles II**.  
Se desarrollaron las pantallas principales de una app de videojuegos (catálogo, detalles, favoritos, perfil) utilizando componentes de React Native como:

- `View`
- `Text`
- `Image`
- `FlatList`
- `TouchableOpacity`
- `StyleSheet`

La aplicación es navegable y visualmente atractiva, simulando una experiencia real.

---

## 📚 Fundamentación Teórica
El proyecto se apoya en:

- **React Native**: Framework para desarrollo móvil multiplataforma.
- **Expo**: Plataforma para crear y probar aplicaciones móviles.
- **Firebase (conceptual)**: Plataforma BaaS para autenticación, base de datos y almacenamiento.
- **Supabase (implementado)**: Base de datos utilizada para almacenar los puntajes obtenidos por los jugadores.

---

## ⚙️ Actividades Desarrolladas
- Configuración del entorno de desarrollo con Expo y React Native.
- Creación de la estructura del proyecto (pantallas, navegación, estilos, componentes, datos).
- Diseño de la interfaz gráfica del juego y pantallas de puntajes.
- Implementación del juego **Caza Insectos** con interacción táctil.
- Manejo de estados con `useState` y procesos automáticos con `useEffect`.
- Configuración de navegación con **Stack Navigator** y **Bottom Tab Navigator**.
- Integración con **Supabase** para almacenamiento de puntajes.
- Pruebas y corrección de errores.
- Documentación del proyecto en GitHub.

## ✅ Resultados
- Se implementó un videojuego interactivo denominado **Caza Insectos**.
- Los usuarios pueden ingresar con un nombre, capturar insectos, acumular puntos y visualizar los mejores resultados.
- Se integró Supabase para registrar puntajes.
- Se configuró correctamente la navegación entre pantallas.
- La aplicación cumplió con los requerimientos planteados inicialmente.

---

## 📌 Conclusiones
- Se aplicaron conocimientos de React Native para crear una aplicación móvil funcional.
- `useState` permitió controlar puntaje, tiempo e insectos disponibles.
- `useEffect` facilitó procesos automáticos como temporizador e inicio del juego.
- Supabase permitió almacenar y consultar resultados externos.
- La navegación mejoró la organización de la aplicación.

---
