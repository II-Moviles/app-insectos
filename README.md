# Caza Insectos 🐞

Aplicación móvil desarrollada con React Native y Expo como parte del taller práctico de Aplicaciones Móviles II.

El objetivo de la aplicación es capturar insectos que aparecen en pantalla, acumular puntos y guardar los mejores puntajes de los jugadores.

## Tecnologías utilizadas

- React Native
- Expo
- TypeScript
- React Navigation
- Supabase
- AsyncStorage

## Funcionalidades

- Registro de jugador mediante nombre.
- Juego interactivo de captura de insectos.
- Contador de tiempo.
- Sistema de puntuación.
- Guardado de resultados en Supabase.
- Visualización de mejores puntajes.
- Navegación mediante Stack Navigator y Bottom Tabs.
